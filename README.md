# Park At A Glance

Get up-to-date ⛅ weather, ⛔ alerts, 📰 news, and 🏕️ camping information for any of America's National Parks. 🌲

## Features

- Search for any American National Park or Monument and get suggestions as you type
- View a general summary and photo of the park with a link to directions
- Get current weather in addition to a weekly forecast
- View current alerts and news articles
- View campground descriptions with links to make reservations

## Technologies Used

- React
- Typescript
- Tailwind
- NPS API
- OpenWeatherMap API
